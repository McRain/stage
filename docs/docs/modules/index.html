Modules ​

Модули - способ расширить возможности. С помощью модулей можно реализовать большинство функциональности:

  • новые типы моделей
  • парсинг форматов
  • визуализация данных получаемых от вашего API.

и многое, многое другое.

Готовые модули ​

В приложении вы можете найти библиотеку модулей (Global Modules)

Меню /Библиотеки/Модули

Выбирайте подходящий модуль и добавляйте в проект. Практически все модули активрируются сразу, добавляя функционал, новые модели, новые исполнители или что то еще.

Модули проекта ​

После добавления модуль появится в проекте. Посмотреть список модулей проекта можно кликнув на кнопке меню

Для создания модуля вам нужно знать совсем немного правил, которые помогут вашему модулю интегрироваться в платформу без проблем.

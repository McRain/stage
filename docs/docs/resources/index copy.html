Ресурсы ​

Для отображения моделей на сцене используются ресурсы - это файлы с 3Д геометрией, изображения, hdr, видеофайлы и другое.

На данный момент платформа поддерживает 3 вида ресурсов:

  • Модель - включает геометрию, материалы, иерархию 3Д объектов. Поддерживаемые форматы: gltf и glb

  • Изображение - Поддерживаемые форматы: png, jpg, webp

  • HDR - файлы, используемые для создания реалистичного освещения и окружения

В большинстве случаев оптимальным вариантом использования является подлная подготовка ресурса во внешних редакторах, и добавление в проект готовых к использованию ресурсов.

Создание ресурсов ​

Вы можете разместить ресурсы на вашем сервере, или в другом доступном месте. При добавлении ресурса в проект потребуется только ссылка на ресурс. Сервер, где вы размещаете ресурсы должен поддерживать CORS к домену stage.reneos.com .

CORS (Cross-Origin Resource Sharing) — это механизм безопасности в браузерах, который использует специальные HTTP-заголовки, чтобы управлять доступом веб-страницы к ресурсам с другого домена. Он существует, чтобы разрешать или запрещать межсайтовые запросы, которые по умолчанию блокируются "Политикой одинакового источника" (Same-Origin Policy).

Если вы используете для хранения ресурсов облачное хранилище - чаще всего его можно настроить для разрешения CORS.

Для Amazon S3: Если ваши ресурсы хранятся в Amazon S3, вам нужно настроить CORS-конфигурацию для bucket через консоль AWS, указав разрешённые origin и методы

После добавления ресурс появится в панели "Ресурсы"

Добавление ресурса в сцену ​

Ресурс может быть преобразован в модель и добавлен в сцену. Для этого можно перетащить ресурс из панели ресурсов в панель моделей, если на основе этого ресурса может быть создана модель - модель появится в панели моделей и на сцене (изначально у модели отключена видимость)

Второй вариант - воспользоваться кнопкой с иконкой справа от имени ресурса в панели ресурсов.

Параметры ресурсов ​

При выборе ресурса в панели ресурсов его параметры будут показаны в панели инспектора. Некоторые параметры можно изменить. Некоторые параметры при изменении влияют в том числе на модели созданные из этого ресурса.
